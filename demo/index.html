saveAsApp

You have to use a mobile device or override you useragent string to view this demo.

Usage:
$.saveAsApp({
    initDelay   : 1500
    closeDelay  : 12000,
    target      : 'body',
    ignoreCookie: 'saveAsAppIgnored',
    showedCookie: 'saveAsAppShowed',
    frequency   : 120000,
    infoTexts   : {
        'ios-safari'    : '',
        'win-ie'        : '',
        'android-native': ''
    }
});
